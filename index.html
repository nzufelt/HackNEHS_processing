Processing Demo

HackNEHS 2017

This site contains some of the basics that you need to get started today. Here's what you need:

  1. Head over to processing.org and download Processing. You might also need to download the Java Development Kit, which lets your computer create and run Java code.
  2. Download the starter file and put it somewhere on your computer where you can find it.
  3. Help a neighbor do the same!

When you're done, you can head over to the github repo to grab the final code!

Happy Hacking!

Other notes:

  • Check out openprocessing.org for other ideas, inspiration, and code!
  • You can host a processing sketch on a website (Like I'm doing above) using processing.js, a JavaScript library.
