Processing Demo

HackNEHS 2017

This site contains some of the basics that you need to get started. Here's what you need:

  1. Head over to processing.org and download Processing.
  2. Download the starter code and put it somewhere on your computer where you can find it.
  3. Help a neightbor do the same!

When you're done, you can head over to the github repo to grab the final code!

Happy Hacking!
